JavaScript Data Types

JavaScript supports different types of data. The main data types are:

  • Number - for numeric values
  • String - for text
  • Boolean - true or false
  • Undefined - a variable without a value
  • Null - an empty or unknown value
  • Object - for collections of data
  • Symbol - for unique identifiers (ES6). Symbols are always unique, even if they have the same description. They are often used as object property keys to avoid property name collisions.
  • BigInt - for large integers (ES2020)

Primitive vs Non-Primitive Types

  • Primitive Types: Number, String, Boolean, Undefined, Null, Symbol, BigInt
  • Non-Primitive (Reference) Types: Object (including Array, Function, etc.)

Primitive types store single values and are immutable.
Non-primitive types store collections or more complex entities and are mutable.
